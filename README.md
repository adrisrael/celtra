# Celtra

A cinematic creator marketplace concept built from scratch with semantic HTML, CSS-generated visuals, and vanilla JavaScript.

The experience includes:

- A seamless four-column creator reel hero
- A browsable and filterable creator marketplace
- Working bag and gift interactions
- A creator early-access application flow
- Responsive navigation and mobile layouts

## Run locally

Open `index.html` directly, or serve the directory with any static file server:

```bash
python3 -m http.server 4173
```
